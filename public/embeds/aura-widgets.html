Using this library

32 widgets in four groups: hero figures (01–08), relational (09–16), metrics & charts (17–22), and scheduling, product & status (23–32). Each tile is captioned with what it is and where it maps.

Every widget is self-contained HTML + CSS with a light vanilla-JS animation layer (count-ups, bar-grows, live timers) that fires once on scroll and is fully prefers-reduced-motion safe. Three families only — apricot, rose, lavender; no gold (that's Finance OS).

This is a reference / preview board for the marketing system — copy from it freely. For the product these become SquircleWidget components mirroring the design-system tokens. Nothing here modifies the read-only design system.
