# Media

Drop the design-system video here as **`overview.mp4`** and it will appear in the video
section (between the hero and Overview). To use a different name or path, update the
`src` on the `VideoPlayer` in `src/showcase/sections/VideoSection.tsx`.

Until a file is present, the player shows an empty placeholder. Once a video is added it
plays inline, and docks to a bottom-right mini-player (picture-in-picture) while playing
as you scroll past.

Recommended: MP4 (H.264), 16:9, ≤ ~1080p for a snappy load.
